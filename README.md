# 🍲 Recipe Recommendation System

A web-based Recipe Recommendation System built using **Flask** and **SQLite**. The application allows users to search for recipes based on available ingredients and instantly displays matching recipes along with ingredients and preparation steps.

## 🚀 Features

* Search recipes by ingredient
* Fast recipe retrieval using SQLite database
* User-friendly web interface
* Displays recipe ingredients and cooking instructions
* Lightweight and easy to deploy

## 🛠️ Tech Stack

* Python
* Flask
* SQLite
* HTML/CSS

## ⚙️ Installation

```bash
pip install flask
```

## ▶️ Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```

## 🎯 How It Works

1. User enters an ingredient.
2. Flask receives the search request.
3. SQLite database is queried for matching recipes.
4. Matching recipes are displayed with ingredients and preparation steps.
